Resumen

AI Copilot System Prompt

Un copiloto llamado "AgTech Mentor" que guía a Jimy como asesor técnico, de negocio, IP y visa. Incluye instrucciones para Claude Sonnet 4.5, modos rápidos y plantillas de seguimiento.

6 modos especializados Prompt listo para copiar Next 3 steps en cada respuesta

Qué incluye

Prompt principal

Rol completo para Claude Sonnet, contexto del proyecto, personalidad y responsabilidades (advisor, architect, PM, IP, visa y coach).

Modos de conversación

Seis modos (Strategic, Technical, Customer, Paper, Visa, Weekly Plan) con instrucciones específicas para cada tipo de pregunta.

Quick prompts

Atajos para onboarding de clientes, preparación de entrevistas, planificación semanal y chequeos mensuales.

Standing instructions

Reglas sobre tono bilingüe, foco en próximas 2 semanas, seguimiento de riesgos y recordatorio constante del objetivo visa.

Cómo usarlo

  1. Entra a claude.ai y abre una conversación nueva.
  2. Copia todo el prompt principal desde la parte 1 del documento e incluye tus datos en los campos entre corchetes.
  3. Arranca con el "Conversation Starter": “Status update + ¿qué priorizo esta semana?”.
  4. Usa los modos rápidos cuando necesites: por ejemplo “Customer Mode” antes de una entrevista o “Visa Mode” para preparar evidencia.
  5. Reserva 20 minutos los viernes para ejecutar el template de retrospectiva mensual y capturar learnings.

Acciones recomendadas

  • Guardar una copia editable del prompt para actualizar datos clave (clientes, hitos, riesgos).
  • Crear un calendario que recuerde los check-ins semanales y mensuales sugeridos.
  • Sincronizar respuestas clave del copiloto con el Plan de Negocio y el Outline de Patente.
